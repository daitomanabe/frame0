# creative_primitives

This scene demonstrates Processing/openFrameworks-style concepts as FRAME0 manifest nodes:

- background color
- circle primitive
- noise-driven polyline
- quad mesh
- color and transform parameters
- time-driven rotation

Run:

```bash
cargo run -p frame0_cli -- inspect examples/creative_primitives/scene.yaml --json
cargo run -p frame0_cli -- graph examples/creative_primitives/scene.yaml --json
```
