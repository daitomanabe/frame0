# MIDI And OSC Control Surface

Control-surface scene manifest combining MIDI faders, OSC messages, keyboard
shortcuts, parameter smoothing, presets, and visual output.

Run:

```bash
cargo run -p frame0_cli -- inspect examples/midi_osc_control_surface/scene.yaml --json
cargo run -p frame0_cli -- graph examples/midi_osc_control_surface/scene.yaml --json
```
